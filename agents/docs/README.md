# Session 3 Drafts

Use this folder for session-specific drafts generated by the coding agent.

- `agents/templates/` holds the blank canonical templates.
- `agents/docs/` holds filled drafts for your review.
- You review and approve these drafts before implementation or commit or merge decisions proceed.

Recommended naming pattern:

- `agents/docs/<task-slug>-01-task-spec.md`
- `agents/docs/<task-slug>-02-implementation-plan.md`
- `agents/docs/<task-slug>-03-review-and-decision.md`
- `agents/docs/<task-slug>-04-hardening-note.md`
